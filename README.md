A comprehensive collection of practical, lightweight Python mini-projects focused on mastering fundamental data types and data structures.

---

## 🧵 01. String Manipulation & Operations (`str`)

This section contains standalone console applications focused on string indexing, slicing, built-in text methods, and character validation.

### 🛠️ Included Projects

#### 1️⃣ Text Formatter (`text_formatter.py`)
A utility script designed to clean and modify raw user text inputs.
*   **Key Python Concepts Used:** `.strip()`, `.upper()`, `.lower()`, `.join()`, `.split()`, and reverse string slicing `[::-1]`.
*   **Features:** Removes leading/trailing whitespaces, toggles letter casing, reverses strings, and generates URL-friendly text slugs.

#### 2️⃣ Text Analyzer (`text_analyzer.py`)
An analysis tool that breaks down text composition metrics and structure.
*   **Key Python Concepts Used:** List comprehensions, `.isdigit()`, `.isalnum()`, membership operators (`in`), and conditional evaluations.
*   **Features:** Returns counts for total characters, word counts, vowels, digits, and white spaces. Automatically flags if the input is a valid palindrome.

#### 3️⃣ Password Strength Checker (`password_strength.py`)
A security evaluation script that tests character arrays against password complexity standards.
*   **Key Python Concepts Used:** Built-in `any()` function, boolean expressions, looping checks (`.isupper()`, `.islower()`), and scoring logic.
*   **Features:** Evaluates a password for optimal length, uppercase/lowercase letters, numerical digits, and special symbols to output a strength rating (Weak, Medium, Strong).
